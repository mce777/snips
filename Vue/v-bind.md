# Binding

Bind attributes via **data** or **props**.

This is **one-way** data-binding (for two-way, see v-model).

```
<!-- full syntax -->
<a v-bind:href="url"> ... </a>

<!-- shorthand -->
<a :href="url"> ... </a>

<!-- shorthand with dynamic argument (2.6.0+) -->
<a :[key]="url"> ... </a>
```
